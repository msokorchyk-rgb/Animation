# Animation

Це невеликий проєкт на p5.js з анімацією персонажа.  
Персонаж має три стани: стояти, крокувати та бігти.  
Анімація зроблена покадрово.  
Можна змінювати масштаб слайдером і перевертати персонажа кнопкою Flip.  
FPS для ходьби і бігу різний, щоб рух виглядав швидшим.  
Hitbox показаний червоним колом навколо тулуба.  
Можна покращити: додати більше кадрів для кожного стану, кольори і деталі.
